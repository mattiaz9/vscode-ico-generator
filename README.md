# ICO Generator

A VS Code extension for generating ICO files from images.

## Features

- Hello World command (placeholder)

## Development

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Open the folder in VS Code and press `F5` to launch the Extension Development Host.

3. In the new window, run the "Hello World" command from the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`).

## Building

```bash
pnpm run compile
```

## Packaging

```bash
pnpm add -g @vscode/vsce
vsce package
```
