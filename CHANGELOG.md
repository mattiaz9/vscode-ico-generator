# Change Log

All notable changes to the ICO Generator extension will be documented in this file.

## [1.0.0] - 2026-03-09

### Added

- Generate ICO files from PNG, JPG, JPEG, and SVG images
- Right-click context menu on supported images in the Explorer
- Command Palette support ("Generate .ico")
- Configurable output size (16–256 pixels)
- Configurable background color (transparent or hex)
- Progress notification during conversion
